# 🏆 OLP & ICPC 2026 Training 

Kho lưu trữ này chứa toàn bộ source code, template và tài liệu trong quá trình mình luyện tập để chuẩn bị cho kỳ thi **Olympic Tin học Sinh viên (OLP)** và **ICPC 2026**.

## 👨‍💻 Thông tin cá nhân
* **Tác giả:** Đỗ Thanh Triết
* **Đơn vị:** Đại học FPT
* **Ngôn ngữ sử dụng:** C++ only

## Cấu trúc Repository

Mã nguồn được tổ chức theo các nền tảng luyện tập và các chủ đề thuật toán cụ thể:

* `VNOI/`: Lời giải cho các bài tập trên VNOI.
* `Codeforces/`: Lời giải các bài tập trên Codeforces.

##  Mục tiêu luyện tập
* Ôn tập và củng cố vững chắc nền tảng Data Structures & Algorithms.
* Tối ưu hóa tốc độ tư duy, implement và debug trong các kỳ thi giới hạn thời gian.
* Đạt thành tích cao nhất có thể tại OLP và ICPC.

## Hướng dẫn biên dịch và chạy thử (Local)

Đối với các bài tập sử dụng **C++**, sử dụng lệnh sau để biên dịch với chuẩn C++17 và tối ưu hóa O3:

```bash
# Biên dịch
g++ -O3 -std=c++17 main.cpp -o main

# Chạy với input/output chuẩn
./main < input.txt > output.txt
